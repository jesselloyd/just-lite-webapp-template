This is a basic modal. No action is bound to its action button. You can quickly mock up modals that don't require functionality, or you can pass functions to it via the action attribute.

The modal comes with built in error detection and has visual indicators in the event of some action failing. This has been implemented with a promise-based system - if your promise is rejected, the modal will interpret this as an error. Try it out with the Show Error Modal Button.

 Click the Error button to view the effect generated when a modal action receives an error. The UI is your friend, use it to alert users when something unexpected happens.
